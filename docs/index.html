📋 Submission Format

  1. Every market listing is a GitHub Discussion in the hydrooj-addons-market repository.
  2. The discussion title becomes the addon name shown in the market.
  3. The discussion body must include at least one of:
    npm: <package-name> — for npm packages
    git: <https://…/repo.git> — for Git repositories
  4. Any other lines in the body become the addon description.
npm: hydrooj-plugin-example git: https://github.com/yourname/hydrooj-plugin-example.git A short description of what this addon does.

✏️ Addon Details

Title is required.
Must be a valid npm package name. At least one of npm or git is required.
Enter a valid npm package name (no spaces, no capital letters).
Must start with https://, http://, or git@ and end with .git.
Must be a valid Git URL ending in .git
