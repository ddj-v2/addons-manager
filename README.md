hydrooj-addons-manager Plugin for HydroOJ
=======================================
This plugin provides an Addons Manager for HydroOJ, allowing users to easily manage and install addons in web for their HydroOJ instance.

Features
--------
- List installed addons
- Add new addons via package name or Git URL
- Remove existing addons
- Update addons to the latest version
- Browse and install addons from the **Addon Market** (powered by GitHub Discussions)

Addon Market Submission
-----------------------
Want to submit your addon to the community market?

Visit the **[Addon Submission Form](https://bryan0324.github.io/hydrooj-addons-manager/)** (GitHub Pages) to fill in your addon details and generate a correctly-formatted GitHub Discussion in one click.

The submission form helps you create a discussion in the correct format:

```
npm: hydrooj-plugin-example
git: https://github.com/yourname/hydrooj-plugin-example.git

A short description of what this addon does.
```

**To enable GitHub Pages** for this repo, go to *Settings → Pages* and set the source to `docs/` on the branch you want to deploy.

hydrooj-addons-manager 套件 
========================
這個套件為 HydroOJ 提供了一個套件管理器，允許使用者在網頁介面中輕鬆管理和安裝 HydroOJ 的套件。

功能
--------
- 列出已安裝的套件
- 透過套件名稱或 Git URL 新增套件
- 移除已存在的套件
- 更新套件到最新版本
- 透過**插件市場**瀏覽並安裝社群套件（由 GitHub Discussions 提供支援）

插件市場提交
-----------------------
想將您的套件提交到社群市場？

請前往 **[插件提交表單](https://bryan0324.github.io/hydrooj-addons-manager/)**（GitHub Pages），填寫套件資訊，一鍵生成格式正確的 GitHub Discussion。
